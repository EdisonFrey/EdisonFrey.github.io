自建节点指南

安装BBR

推荐使用一键脚本

https://cdn.statically.io/gh/LKSIUS/SH/main/bbr.sh

解析域名

在Cloudflare中添加DNS，将你的服务器地址与域名地址绑定，并在将SSL/TLS中加密模式改为完全。

安装WARP

非IPV6 only的机器请跳过这一步。IPV6 only的机器，要想用来科学上网就需要通过安装WARP补全IPV4，以获取完整的上网体验。推荐使用fscarmen的WARP脚本，项目地址见GitLab。

wget -N https://gitlab.com/fscarmen/warp/-/raw/main/menu.sh && bash menu.sh [option] [lisence/url/token]

wget -N https://gitlab.com/fscarmen/warp/-/raw/main/warp-go.sh && bash warp-go.sh [option] [lisence]

Warp和Warp-go选择其一，按照引导安装，添加IPV4代理或添加双栈代理。

安装Xray面板

推荐使用X-UI，方便管理，项目地址见Github。记住端口、用户名及密码。

bash <(curl -Ls https://raw.githubusercontent.com/vaxilu/x-ui/master/install.sh)

docker run -d \
    --name x-ui \
    --restart always \
    --network host \
    -v /opt/x-ui/:/etc/x-ui \
    -v /opt/x-ui/cert:/root/cert \
    -v /opt/cert:/opt/cert \
    tzebit/x-ui:latest

docker run -d \
    --name x-ui \
    --restart always \
    --network host \
    -v /opt/x-ui/:/etc/x-ui \
    -v /opt/x-ui/cert:/root/cert \
    -v /opt/cert:/opt/cert \
    registry.ipv6.docker.com/tzebit/x-ui:latest

创建节点

不出意外的话，你现在已经可以通过http://[ip]:端口访问X-UI了，打开Xray面板，输入账号密码，登录面板，点击入站列表，添加，先将传输改为ws，再将协议改为trojan，端口设置为8443，flow为无，证书方式选择certificate file content，公钥内容和私钥内容如下，其余默认或留空即可。点击详细信息，查看，复制链接，导入软件中，由于该项目长时间未更新，因此你需要在软件中将节点的传输协议由tcp改为ws，地址为IP即为直连，为域名即为Cloudflare CDN。
